## [1.0.0] - 2026-01-15

### Added
- **Rebranded to TunTun Extension** from BlockTube fork
- Family-focused messaging and documentation
- Parent-oriented Quick Start Guide in README

- CHANGELOG.md for tracking changes

### Changed

- Version bumped to 1.0.0 (major release)

### Inherited from BlockTube v0.4.7
All core features from BlockTube are included:
- Block videos by title, channel name, channel ID, video ID
- Block comments by user or content
- Context menu blocking
- Mobile YouTube support (m.youtube.com)
- Password protection for settings
- Pre-rendering filters for performance
- Regex and JavaScript filtering
- Duration-based filtering
- Block YouTube Shorts, Movies, Trending
- Custom block messages

### Notes
- Extension code remains GPL v3 (inherited from BlockTube)
- Prominent BlockTube attribution maintained in README and code
- Premium features (cloud sync, remote management) planned for future releases
